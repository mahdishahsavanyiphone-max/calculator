# calculator

Description:
A simple calculator application built with Python and Tkinter that supports basic arithmetic operations, square root, power calculations, decimal numbers, and editing functions such as clear and backspace

featuer:
- Perform addition, subtraction, multiplication, and division
- Calculate square roots
- Calculate powers
- Clear the current expression
- Delete the last entered character (Backspace)
- Support decimal numbers

Technologies Used:
- Python
- Tkinter (GUI)
